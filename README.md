# SampleList
Sample app that fetches data from https://fetch-hiring.s3.amazonaws.com/hiring.json API after checking for network connectivity.

Uses:
Retrofit,
LiveData,
MVVM,
OKHTTP3,
Coroutines, 
Kotlin
